实验卡片标准¶

每张实验卡片应包含：

区块 内容要求
假设 改动是什么，为什么可能提升 recall
设计 方法、数据切分、候选/排序流程，必要时加图
输入 dataset version、split version、config path
命令 可复现命令或 notebook 入口
输出 指标、产物路径、图表
分析 哪个 target 提升，原因是什么
决策 采用、拒绝、延后
